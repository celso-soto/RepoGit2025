Curso Git & GitHub - Itgrarte 2025

Clase 1: Git y repositorios

En esta primera clase, exploraremos los fundamentos de Git, el sistema de control de versiones distribuido más popular. Aprenderás a inicializar un repositorio, entender su estructura básica y cómo realizar tus primeros "commits" para guardar el historial de tus proyectos.

Clase 2: Ramas, merge y rebase

La segunda clase se centrará en el manejo de ramas, una característica esencial de Git que permite trabajar en diferentes funcionalidades o correcciones de errores de forma aislada. Cubriremos cómo crear, cambiar y eliminar ramas, así como las estrategias de "merge" y "rebase" para integrar cambios.

Clase 3: El área de trabajo "stash" y actualizar ramas

En la tercera sesión, profundizaremos en el comando "stash", una herramienta útil para guardar temporalmente cambios no confirmados cuando necesitas cambiar de rama rápidamente. También veremos cómo mantener tus ramas locales actualizadas con los últimos cambios de otras ramas o del repositorio remoto.

Clase 4: Crear cuenta en GitHub y clonar repositorios

Esta clase te guiará en el proceso de crear tu cuenta en GitHub, la plataforma de alojamiento de repositorios más grande del mundo. Aprenderás a clonar repositorios existentes de GitHub a tu máquina local para comenzar a colaborar en proyectos o explorar código de otros usuarios.

Clase 5: GitHub Pages y Pull Request

Finalmente, en la quinta clase, descubriremos cómo usar GitHub Pages para publicar sitios web directamente desde tus repositorios. También exploraremos el concepto de "Pull Request", una característica fundamental para proponer cambios y colaborar de manera efectiva en proyectos de código abierto.
